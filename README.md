# Advanced SQL Analytics

This repository contains advanced SQL practice and class exercises focused on analytical querying techniques. It showcases skills in joins, subqueries, filtering, aggregations, and window functions using structured SQL scripts.

## Skills Demonstrated
- SQL query development
- INNER JOIN, LEFT JOIN, and multi-table joins
- Subqueries and nested logic
- Aggregations and grouping
- Window functions
- Analytical problem solving
- Query organization and documentation
## Key Concepts Covered
- Filtering and sorting query results
- Multi-table joins and relational logic
- Nested subqueries for complex analysis
- Aggregations and grouping techniques
- Window functions (RANK, ROW_NUMBER, PARTITION BY)
- Writing structured, readable SQL queries

## Tools & Environment
- SQL (MySQL / PostgreSQL / course environment)
- Relational database concepts
- Query-based analytical workflows

## Portfolio Value
This repository demonstrates practical SQL skills used in real-world analytics workflows, including data exploration, reporting, and business intelligence tasks. It highlights the ability to write efficient queries, structure analytical logic, and organize technical work in a clean, professional format.
